Computer Network Topology: Detailed Guide

Jan 02, 2026 | Networking

Network Topology কী?

Network Topology হলো একটি Network-এর Device (Node) এবং সংযোগগুলোর (Link) জ্যামিতিক বিন্যাস। এটি নির্ধারণ করে যে কীভাবে Computer, Server এবং অন্যান্য Device-গুলো একে অপরের সাথে যুক্ত এবং কীভাবে তাদের মধ্যে Data প্রবাহিত হয়। Topology মূলত দুই প্রকার:

  • Physical Topology: Cable এবং Device-এর প্রকৃত শারীরিক বিন্যাস।
  • Logical Topology: শারীরিক বিন্যাস যাই হোক না কেন, Data কীভাবে Network-এর মাধ্যমে প্রবাহিত হয়।

Topology-র Classification (Types of Topologies)

GeeksforGeeks-এর Information অনুযায়ী প্রধান Topology-গুলো নিচে Detailed Discussion করা হলো:

1. Point-to-Point Topology

এটি সবচেয়ে সহজ যোগাযোগ Method যেখানে দুটি Node-এর মধ্যে একটি সরাসরি Link থাকে। একজন Sender এবং একজন receiver থাকে। এটি High Bandwidth প্রদান করে।

2. Mesh Topology (মেশ Topology)

মেশ Topology-তে প্রতিটি Device অন্য প্রতিটি Device-এর সাথে সরাসরি যুক্ত থাকে।

  • Link সংখ্যা: $N$ সংখ্যক Device-এর জন্য মোট Link প্রয়োজন $N(N-1)/2$।
  • Advantage: অত্যন্ত Reliable (Robust), Fast যোগাযোগ এবং High Security। একটি Link নষ্ট হলেও পুরো Network সচল থাকে।
  • অ Advantage: Installation কঠিন এবং Cabling Cost অনেক High।

3. Star Topology (স্টার Topology)

এখানে সব Device একটি কেন্দ্রীয় Hub বা Switch-এর সাথে যুক্ত থাকে।

  • Advantage: সহজে Setup করা যায়, একটি Link নষ্ট হলে অন্যগুলো প্রভাবিত হয় না, Fault শনাক্ত করা সহজ।
  • অ Advantage: কেন্দ্রীয় Hub নষ্ট হলে পুরো Network অচল হয়ে পড়ে। Hub-এর Performance-এর ওপর পুরো Network নির্ভরশীল।

4. Bus Topology (বাস Topology)

সব Device একটি সাধারণ Backbone Cable-এর সাথে যুক্ত থাকে। এটি দ্বিমুখী (Bi-directional) যোগাযোগ সমর্থন করে।

  • Advantage: ছোট Network-এর জন্য সস্তা এবং সহজ।
  • অ Advantage: মূল Cable নষ্ট হলে পুরো Network অচল হয়। Traffic High হলে Collision বাড়ে। Security কম।

5. Ring Topology (রিং Topology)

প্রতিটি Device তার পাশের দুটি Device-এর সাথে যুক্ত হয়ে একটি বৃত্ত বা Ring তৈরি করে। এখানে 'Token Passing' Method ব্যবহার করা হয়।

  • Advantage: Data Collision কম হয়, Installation Cost Relatively low।
  • অ Advantage: যেকোনো একটি Node নষ্ট হলে পুরো Network অচল হয়ে যায়। Troubleshooting কঠিন।

6. Tree Topology (Tree Topology)

এটি Star Topology-র একটি Advanced রূপ যা হায়ারার্কিকাল (Hierarchical) Structure অনুসরণ করে। একটি Main Hub-এর সাথে অনেকগুলো Secondary Hub যুক্ত থাকে।

  • Advantage: বড় Network-এর জন্য উপযোগী, নতুন Device যোগ করা সহজ।
  • অ Advantage: Main Hub নষ্ট হলে নিচের সব শাখা অচল হয়ে যায়। Cabling Cost High।

7. Hybrid Topology (হাইব্রিড Topology)

দুই বা ততোধিক ভিন্ন ধরণের Topology-র সমন্বয়ে এটি তৈরি হয়। যেমন- স্টার এবং বাস Topology-র মিশ্রণ।

  • Advantage: অত্যন্ত Flexible (Flexible) এবং বড় Network-এর Scope বাড়ানো সহজ।
  • অ Advantage: Design করা জটিল এবং Maintenance Cost High।

Topology-র গুরুত্ব

  • Performance: সঠিক Topology Selection করলে Network-এর Speed বৃদ্ধি পায়।
  • Reliable তা: Mesh বা Star Topology Backup সংযোগ প্রদান করে Reliable তা বাড়ায়।
  • বিস্তৃতি: সঠিক Plan থাকলে ভবিষ্যতে Network বাড়ানো সহজ হয়।
  • Security: Topology অনুযায়ী Data প্রবাহ নিয়ন্ত্রণ করে Advanced Security নিশ্চিত করা সম্ভব।

Summary: Network-এর Size, Budget এবং প্রয়োজনীয়তার ওপর ভিত্তি করে সঠিক Topology Selection করা অত্যন্ত জরুরি। Modern Corporate Network-এ সাধারণত Star বা Hybrid Topology বেশি ব্যবহৃত হয়।
