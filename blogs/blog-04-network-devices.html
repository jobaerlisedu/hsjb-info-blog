Network Devices Details - Hub, Switch, Router, Gateway

31 December, 2025 | Networking

Introduction: Importance of Network Devices

Network Device হলো সেই সব Hardware সরঞ্জাম যা একটি Computer Network-এর বিভিন্ন Device-এর মধ্যে Data আদান-প্রদান, সংযোগ স্থাপন এবং Security নিশ্চিত করতে ব্যবহৃত হয়। এগুলো ছাড়া একটি Modern Network কল্পনা করা অসম্ভব।

Network Device-এর 5টি প্রধান কাজ:

  • Data Transmission (Transmission): এক Device থেকে অন্য Device-এ Information পাঠানো।
  • Communication Efficiency (Efficiency): Fast তম Path ে Data পাঠিয়ে Network-এর জট কমানো।
  • Performance বৃদ্ধি (Performance): Traffic Management-এর মাধ্যমে Speed বাড়ানো।
  • Security নিশ্চিত করা (Security): অননুমোদিত প্রবেশ রোধ করা এবং Security Rule প্রয়োগ করা।
  • Coverage বিস্তার (Coverage): Signal শক্তিশালী করে Network-এর সীমানা বাড়ানো।

1. Physical Layer (Layer 1) Device

এই Device-গুলো সরাসরি বৈদ্যুতিক বা Optical Signal (Raw bits) নিয়ে Task করে। এরা কোনো Addressing বোঝে না।

Hub (Hub)

Hub হলো একটি Multi-port Device যা একটি Local Network-এর সব Device-কে যুক্ত করে। এটি Incoming Signal-কে সব Port-এ Broadcast করে দেয়।

  • Classification:
    • Passive Hub: Signal শুধু পাস করে দেয়, কোনো শক্তিশালী (Regeneration) করে না।
    • Active Hub: এটি রিপিটারের মতো Task করে Signal শক্তিশালী করে।
    • Intelligent Hub: এতে Network ম্যানেজমেন্ট ও মনিটরিং Advantage থাকে।
  • Advantage: সহজ সেটআপ এবং অত্যন্ত সাশ্রয়ী।
  • অ Advantage: Data জট বা Collision Domain High, সিকিউরিটি কম এবং ব্যান্ডউইডথ অপচয় হয়।

রিপিটার (Repeater)

দীর্ঘ Distance ে Data যাওয়ার সময় Signal Weak বা 'Attenuation' হয়। Repeater সেই Weak Signal গ্রহণ করে Noise সরিয়ে ফেলে এবং Signal-কে Bit-by-bit পুনরায় তৈরি (Regenerate) করে পাঠায়।

  • Advantage: Network-এর শারীরিক Scope বাড়াতে সাহায্য করে।
  • অ Advantage: এটি Traffic ফিল্টার করতে পারে না এবং হালকা Network Delay (Delay) তৈরি করে।

Modem (Modem)

Modem (Modulator-Demodulator) Digital এবং Analog Signal-এর মধ্যে রূপান্তরকারী হিসেবে Task করে।

  • কাজ: Computer-এর Digital Signal Analog ে (Modulation) এবং Analog Signal Digital ে (Demodulation) রূপান্তর করা।

2. Data Link Layer (Layer 2) Device

এই Layer-এর Device-গুলো MAC Address (Physical Address) ব্যবহার করে অনেক High বুদ্ধিমত্তার সাথে Task করে।

Switch (Switch)

Switch-কে বলা হয় 'Intelligent Hub'। এটি প্রতিটি Device-এর MAC Address শিখে নিজের একটি Forwarding Table তৈরি করে। এটি Data Unicast বা সরাসরি Destination Port-এ পাঠায়।

  • Classification: Unmanaged (বেসিক), Managed (VLAN ও Security Support করে), এবং Layer 3 Switch (Routing Advantage সহ)।
  • Advantage: Efficient Traffic Management, Collision মুক্ত Network (Dedicated collision domain per port)।
  • অ Advantage: Hub-এর চেয়ে দামি এবং কনফিগার করা কিছুটা জটিল হতে পারে।

Bridge (Bridge)

Bridge দুটি LAN Segment-কে যুক্ত করে একটি Unified Network তৈরি করে। এটি MAC Address Table ব্যবহার করে Local Traffic Filter করে।

  • Classification: Transparent Bridge (অটোম্যাটিক) এবং Source Routing Bridge (ম্যানুয়াল Source Control)।
  • Advantage: Network-এর জট কমানো এবং Logical Scope বাড়ানো।
  • অ Advantage: Processing Time-এর কারণ ে Hub-এর চেয়ে কিছুটা SlowSpeed-র।

Network Interface Card (Network Interface Card - NIC)

NIC Card হলো সেই Hardware যা প্রতিটি Computer বা Smartphone-এ থাকে। এটি Network Cable বা Wi-Fi সংযোগের সরাসরি মাধ্যম। এর ভেতরেই Device-এর স্থায়ী MAC Address থাকে। এতে Internal বা External উভয় প্রকার থাকে।

3. Network Layer (Layer 3) Device

এরা IP Address (Logical Address) নিয়ে Task করে এবং বিভিন্ন Network-এর মধ্যে সংযোগ স্থাপন করে।

Router (Router)

Router হলো Network-এর 'Traffic Police'। এটি বিভিন্ন Network-এর (যেমন: LAN ও Internet) মধ্যে সংযোগ ঘটায়। এটি Routing Table এবং Protocol (RIP, OSPF) ব্যবহার করে Data-র জন্য সেরা Path খুঁজে বের করে।

  • Advantage (Advantages): Broadcast Traffic বন্ধ করে Security বাড়ায় এবং Separate Network-কে যুক্ত করে।
  • অ Advantage (Disadvantages): প্রতিটি Packet Analysis করতে হয় বলে কিছুটা SlowSpeed এবং দাম High।

ব্ Router (Brouter)

এটি Bridge এবং Router-এর সংমিশ্রণ। এটি Routable Protocol-এর জন্য Router হিসেবে এবং Non-routable Protocol-এর জন্য Bridge হিসেবে একই সাথে Task করতে পারে।

4. Upper Layer (Layer 4-7) Device

Gateway (Gateway)

Gateway হলো এক Network থেকে অন্য Network-এর প্রবেশদ্বার। এটি ভিন্ন ভিন্ন Architecture বা Protocol Translator হিসেবে Task করে। এটি একটি Packet-এর Encapsulation Change করে ভিন্ন Format-এ রূপান্তর করে।

Firewall (Firewall)

Firewall হলো Network-এর Gate-keeper বা রক্ষী। এটি Security Rule অনুযায়ী Incoming এবং Outgoing Traffic Monitor ও Block করে। এটি Hardware (Physical appliance) বা Software (Server-based) হতে পারে। এটি Packet Filtering এবং Stateful Inspection-এর মাধ্যমে Security নিশ্চিত করে।
