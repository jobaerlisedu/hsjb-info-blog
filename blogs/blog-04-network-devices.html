Network Devices Explained - Hub, Switch, Router, Gateway

Networking December 31, 2025 Jobaer Hossain

Introduction

Network Devices (also known as networking hardware) are physical devices required for communication and interaction between devices on a computer network. They act as intermediaries that facilitate data transfer across networks.

These devices operate at different layers of the OSI model and perform specific functions to ensure efficient and secure data transmission.

Types of Network Devices

1. Hub

A hub is a basic networking device that connects multiple Ethernet devices, making them act as a single network segment.

  • Functions: Broadcasts data to all connected devices, operates at Physical Layer (Layer 1)
  • Characteristics: No intelligence, creates collisions, half-duplex

2. Repeater

A repeater regenerates or amplifies signals to extend the distance of a network.

  • Functions: Signal regeneration, overcomes attenuation
  • Characteristics: Operates at Physical Layer (Layer 1)

3. Bridge

A bridge connects two separate network segments and filters traffic based on MAC addresses.

  • Functions: Reduces unnecessary traffic, segments network, learns MAC addresses
  • Characteristics: Operates at Data Link Layer (Layer 2)

4. Switch

A switch is an intelligent device that connects devices within a network and forwards data only to the intended recipient.

  • Functions: MAC address learning, frame forwarding/filtering, full-duplex communication
  • Characteristics: Operates at Data Link Layer (Layer 2), reduces collisions

5. Router

A router connects different networks and directs data packets between them using IP addresses.

  • Functions: Path determination, packet forwarding, network interconnection
  • Characteristics: Operates at Network Layer (Layer 3), uses routing tables

6. Gateway

A gateway connects networks using different protocols and acts as a translator between them.

  • Functions: Protocol conversion, connects incompatible networks
  • Characteristics: Operates up to Application Layer (can work at multiple layers)

Comparison Table

Device OSI Layer Function Addressing Used Key Feature
Hub Physical (Layer 1) Broadcasts to all ports None Multi-port repeater
Repeater Physical (Layer 1) Signal amplification None Extends cable length
Bridge Data Link (Layer 2) Filters by MAC MAC Address Segments network
Switch Data Link (Layer 2) Intelligent forwarding MAC Address High performance LAN
Router Network (Layer 3) Routes between networks IP Address Connects different networks
Gateway All layers (up to 7) Protocol translation Varies Connects dissimilar networks

Summary

Each network device serves a specific purpose at different layers of the network stack. Understanding their roles helps in designing efficient, scalable, and secure networks.

  • Use hubs/repeaters only for simple, small networks (rare today)
  • Switches are standard for modern LANs
  • Routers are essential for Internet connectivity
  • Gateways enable communication between entirely different network types
