Network Devices Explained - Hub, Switch, Router, Gateway

Dec 31, 2025 | Networking

Introduction

Network Devices are physical hardware required for interaction between devices on a network. They operate at different layers of the OSI model.

Types of Network Devices

1. Hub

A basic device connecting multiple devices. It broadcasts data to all ports (Layer 1).

2. Repeater

Regenerates signals to extend network distance (Layer 1).

3. Bridge

Connects network segments and filters traffic using MAC addresses (Layer 2).

4. Switch

Intelligent device that forwards data only to the intended recipient (Layer 2).

5. Router

Connects different networks and directs packets using IP addresses (Layer 3).

6. Gateway

Connects networks with different protocols, acting as a translator (All Layers).

Comparison Table
