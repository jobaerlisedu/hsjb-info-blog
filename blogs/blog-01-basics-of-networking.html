Basics of Computer Networking

31 December, 2025 | Networking

Introduction: What is a Computer Network?

Computer Network হলো একে অপরের সাথে যুক্ত একদল Device-এর সমষ্টি, যা Communication এবং Resource Share করার জন্য Task করে। এটি Data আদান-প্রদান এবং Modern Technology-র মেরুদণ্ড হিসেবে Task করে।

  • Connectivity: Computer, Smartphone এবং Server-এর মধ্যে সংযোগ স্থাপন।
  • Resource Sharing: Hardware (Printer, Scanner) এবং Software/File Sharing।
  • Communication: Email, Instant Messaging এবং Video Call-এর Advantage।

Basic Terminology

  • Network: একদল যুক্ত Computer এবং Device যা একে অপরের সাথে Communication ও Data Share করতে পারে।
  • Node: Network-এর এমন যেকোনো Device যা Data পাঠাতে, গ্রহণ করতে বা Forward করতে পারে। এর মধ্যে Laptop, Mobile, Printer, Earbud, Server ইত্যাদি রয়েছে।
  • Networking Device: যেসব Device Network Operation ও সমর্থন করে। যেমন- Router, Switch, Hub এবং Access Point।
  • Transmission Media: যে Physical বা Wireless মাধ্যমে র ভেতর দিয়ে এক Device থেকে অন্য Device-এ Data Flow করে।
    • Wired Media: Ethernet Cable, Optical Fiber
    • Wireless Media: Wi-Fi, Bluetooth, Infrared
  • Service Provider Network: বাইরের যেসব প্রতিষ্ঠান বা Provider (যেমন- ISP বা Internet Service Provider, Mobile Operator) User-দের Network ব্যবহারের স Capacity ইজারা বা Lease দেয়।

1. How does a Computer Network work?

একটি Network-এর প্রধান ভিত্তি হলো Node (Nodes) এবং Link (Links)।

  • Node: Network-এর যেকোনো Device যা Data পাঠায় বা গ্রহণ করে (যেমন: Laptop, Router, Smartphone)।
  • Link: সেই মাধ্যম যা Node-গুলোকে যুক্ত করে (যেমন: Ethernet Cable বা Wi-Fi Signal)।
  • Protocols: Network-এর Device-গুলো কিছু নির্দিষ্ট Rule মেনে চলে যাতে তারা একে অপরের Language বুঝতে পারে।
  • Firewall: একটি Security Tool যা Network-এর Traffic Monitor করে এবং Unauthorized Access ঠেকায়।

2. Types of Network Architecture

প্রতিটি Network কীভাবে বিন্যস্ত হবে তার ওপর ভিত্তি করে একে দুই ভাগে ভাগ করা যায়:

  • Client-Server Architecture: এখানে একটি শক্তিশালী Central 'Server' থাকে যা অন্য সব 'Client' Device-এর অনুরোধ পূরণ করে।
  • Peer-to-Peer (P2P) Architecture: এখানে কোনো Central Server থাকে না; প্রতিটি Device সমান Capacity-র অধিকারী এবং একে অপরের সাথে সরাসরি Data Share করতে পারে।

3. Essential Network Devices

Device Name Role
Router Connects multiple networks and sends data in the correct path.
Switch Connects devices within a network perfectly.
Gateway Acts as a translator between two completely different types of networks.
Modem Converts digital data to analog signal for internet connection.
Access Point Converts wired network to Wi-Fi or wireless connections.

4. Main Goals and Uses of Computer Network

Networking শুধু Device সংযোগ নয়, এটি Task-এর দক্ষতা এবং সুযোগ বহুগুণ বাড়িয়ে দেয়:

  • Resource Sharing: এক বা Multiple Printer, Scanner এবং File-এর মাধ্যমে Cost কমানো।
  • Remote Access: আপনি পৃথিবীর যেকোনো প্রান্তে বসে আপনার Office-র বা বাসার Computer-এর File-গুলো ব্যবহার করতে পারেন।
  • E-commerce & Payments: Secure Online Shopping এবং কয়েক ক্লিকেই Money লেনদেনের স Capacity।
  • Education & Research: অনলাইনে Class করা, নতুন নতুন Research Paper পড়া এবং পৃথিবীর সেরা সব Mentor-দের সাথে সরাসরি Communication করা।
  • Real-time Collaboration: Multiple মানুষ একসাথে একটি ফাইলে Task করা এবং Task-এর Update সাথে সাথে একে অপরকে জানানো।

Characteristics of a Computer Network

  • Security: Firewall এবং Encryption-র মাধ্যমে সুরক্ষা নিশ্চিত করা।
  • Reliability: Backup এবং Redundancy-র মাধ্যমে সবসময় Network সচল রাখা।
  • Scalability: Performance না কমিয়ে Network বড় করার Capacity।
  • High Performance: Fast Data Transfer এবং কম Latency।
  • Quality of Service (QoS): গুরুত্বপূর্ণ Data-কে অগ্রাধিকার দেওয়া।
