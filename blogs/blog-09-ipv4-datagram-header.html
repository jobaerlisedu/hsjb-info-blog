IPv4 Datagram Header Explained

Jan 02, 2026 | Networking

Introduction

The IPv4 header defines packet routing and delivery. It is typically 20 bytes long.

Header Fields

Version (4 bits)

Set to 4 for IPv4.

IHL (4 bits)

Header length. Min 5 (20 bytes), Max 15 (60 bytes).

Type of Service (8 bits)

Quality of Service parameters (DSCP/ECN).

Total Length (16 bits)

Packet size (Header + Data). Max 65,535 bytes.

Identification, Flags, Fragment Offset

Used for fragmentation control.

TTL (8 bits)

Time To Live. Decrements at each router. Prevents loops.

Protocol (8 bits)

Next protocol (TCP=6, UDP=17, ICMP=1).

Header Checksum (16 bits)

Error checking for the header.

Source & Destination IP (32 bits each)

Sender and Receiver addresses.
