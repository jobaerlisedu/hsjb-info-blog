আইপি অ্যাড্রেস কী? (What is an IP Address?)

ফেব্রুয়ারি ২৭, ২০২৬ | Networking

আইপি অ্যাড্রেস (Internet Protocol Address) হলো একটি ইউনিক গাণিতিক লেবেল যা কোনো কম্পিউটার নেটওয়ার্কে যুক্ত থাকা প্রতিটি ডিভাইসের পরিচয় হিসেবে ব্যবহৃত হয়। সহজ কথায়, এটি আপনার ডিভাইসের একটি ডিজিটাল বাড়ির ঠিকানা, যার মাধ্যমে ইন্টারনেটে তথ্য আদান-প্রদান করা সম্ভব হয়।

আইপি অ্যাড্রেস মূলত দুটি প্রধান কাজ করে:
  • ১. নেটওয়ার্কে ডিভাইসটিকে শনাক্ত করা।
  • ২. ডিভাইসের অবস্থান খুঁজে বের করে অন্য ডিভাইসের সাথে যোগাযোগ স্থাপন করা।

আইপি অ্যাড্রেসের উপাদানসমূহ (Components)

একটি আইপি অ্যাড্রেস প্রধানত তিনটি অংশে বিভক্ত থাকে:

  • Network Portion: এটি ডিভাইসটি কোন নেটওয়ার্কের অন্তর্ভুক্ত তা শনাক্ত করে।
  • Host Portion: এটি ওই নেটওয়ার্কের নির্দিষ্ট ডিভাইসটিকে শনাক্ত করে।
  • Subnet Mask: এটি নির্ধারণ করে আইপি অ্যাড্রেসের কতটুকু অংশ নেটওয়ার্কের এবং কতটুকু হোস্টের।

আইপি অ্যাড্রেসের প্রকারভেদ

১. পাবলিক বনাম প্রাইভেট আইপি

Public IP Address

ইন্টারনেটে সরাসরি অ্যাক্সেস করার জন্য এটি ব্যবহৃত হয়। বিশ্বজুড়ে এটি ইউনিক হয় এবং ISP (Internet Service Provider) এটি প্রদান করে।

Private IP Address

এটি লোকাল নেটওয়ার্কের (যেমন- আপনার বাড়ি বা অফিস) ভেতরে যোগাযোগের জন্য ব্যবহৃত হয়। এটি ইন্টারনেটে সরাসরি রাউটযোগ্য নয়।

২. IPv4 বনাম IPv6

IPv4

এটি ৩২-বিট অ্যাড্রেসিং সিস্টেম, যা ডট দিয়ে আলাদা করা ৪টি সংখ্যার সেট (যেমন: 192.168.1.1)। এতে প্রায় ৪ বিলিয়ন অ্যাড্রেস সম্ভব।

IPv6

আইপি সংকটের কারণে এটি তৈরি করা হয়েছে। এটি ১২৮-বিট অ্যাড্রেসিং সিস্টেম এবং হেক্সাডেসিমাল সংখ্যায় প্রকাশ করা হয়।

৩. স্ট্যাটিক বনাম ডায়নামিক আইপি

  • Static IP: এটি স্থায়ীভাবে কোনো ডিভাইসে বরাদ্দ করা হয় (যেমন সার্ভারের জন্য)।
  • Dynamic IP: এটি সাময়িকভাবে DHCP সার্ভার দ্বারা বরাদ্দ করা হয় এবং নিয়মিত পরিবর্তিত হতে পারে।

৪. কার্যের ভিত্তিতে আইপি অ্যাড্রেস

Unicast Address

একজন প্রেরক থেকে একজন নির্দিষ্ট প্রাপকের কাছে ডেটা পাঠানো। এটি সবচেয়ে সাধারণ যোগাযোগ পদ্ধতি।

Broadcast Address

একটি ডিভাইসের মাধ্যমে নেটওয়ার্কের সকল ডিভাইসে মেসেজ পাঠানো। IPv6-এ এটি ব্যবহৃত হয় না।

Multicast Address

একই সাথে একাধিক নির্দিষ্ট ডিভাইসের গ্রুপে ডেটা পাঠানো (যেমন: লাইভ স্ট্রিমিং)।

Anycast Address

একই আইপি শেয়ার করা গ্রুপ থেকে ভৌগোলিকভাবে সবচেয়ে কাছের ডিভাইসে তথ্য পাঠানো।

IPv4 অ্যাড্রেসের ক্লাসসমূহ (Classes)

সহজে ব্যবস্থাপনার জন্য IPv4 অ্যাড্রেসকে ৫টি ক্লাসে ভাগ করা হয়েছে:

ক্লাস আইপি রেঞ্জ ব্যবহারের ক্ষেত্র
Class A 1.0.0.0 — 127.255.255.255 বিশাল নেটওয়ার্ক (যেমন: মাল্টিন্যাশনাল কোম্পানি)।
Class B 128.0.0.0 — 191.255.255.255 মাঝারি আকারের নেটওয়ার্ক।
Class C 192.0.0.0 — 223.255.255.255 ছোট ব্যবসা বা হোম নেটওয়ার্ক।
Class D 224.0.0.0 — 239.255.255.255 মাল্টিকাস্ট গ্রুপের জন্য সংরক্ষিত।
Class E 240.0.0.0 — 255.255.255.255 গবেষণা ও ভবিষ্যতের ব্যবহারের জন্য।

বিশেষ আইপি অ্যাড্রেস (Special IP Addresses)

  • Loopback Address / Localhost: ১৮৭.০.০.১ (127.0.0.1) অ্যাড্রেসটি নিজের ডিভাইসের কানেক্টিভিটি টেস্ট করতে ব্যবহৃত হয়।
  • Broadcast Address: একটি নেটওয়ার্কের সকল ডিভাইসে তথ্য পাঠাতে ব্যবহৃত হয় (যেমন: 192.168.1.255)।
  • 0.0.0.0: এটি একটি নন-রাউটেবল অ্যাড্রেস যা ডিফল্ট রুট হিসেবে ব্যবহৃত হয়।

আইপি অ্যাড্রেস কীভাবে কাজ করে?

ইউনিক আইডেন্টিফিকেশন

প্রতিটি ডিভাইসে একটি আইপি থাকায় রাউটার তথ্য পাঠানোর সময় সঠিক গন্তব্য চিনতে পারে।

ডেটা প্যাকেটিং

যখন কোনো ডেটা পাঠানো হয়, তখন সেটি ছোট ছোট প্যাকেটে বিভক্ত হয়। প্রতিটি প্যাকেটে সেন্ডার এবং রিসিভারের আইপি থাকে।

রাউটিং (Routing)

রাউটারগুলো আইপি অ্যাড্রেস দেখে নির্ধারণ করে কোন পথে ডেটা পাঠালে দ্রুত পৌঁছাবে।

NAT (Network Address Translation)

আপনার লোকাল ডিভাইসের প্রাইভেট আইপিকে রাউটার পাবলিক আইপিতে রূপান্তর করে ইন্টারনেটে পাঠায়।

বাস্তব উদাহরণ: ইমেইল পাঠানো

নিউইয়র্ক থেকে টোকিওতে একজন বন্ধুকে ইমেইল পাঠানোর ধাপগুলো:

  1. আপনার ইমেইল ক্লায়েন্ট ডেটাকে প্যাকেটে ভাগ করে এবং তাতে আপনার ও আপনার বন্ধুর মেইল সার্ভারের আইপি যুক্ত করে।
  2. আপনার রাউটার প্যাকেটগুলোকে ISP-র কাছে পাঠায়।
  3. রাউটারগুলো গ্লোবাল নেটওয়ার্কের মাধ্যমে প্যাকেটগুলোকে জাপানে টোকিও সার্ভারের কাছে পৌঁছে দেয়।
  4. টোকিও মেইল সার্ভার প্যাকেটগুলোকে একত্র করে এবং আপনার বন্ধুর ডিভাইসে ইমেইলটি পৌঁছে দেয়।

আইপি অ্যাড্রেস চেক করার উপায়

  • Windows: Command Prompt অপেন করে `ipconfig` লিখে এন্টার দিন।
  • Mac/iPhone: Network Settings বা Wi-Fi সেটিংস থেকে আইপি চেক করা যায়।
  • Public IP: গুগল সার্চে "What is my IP" লিখলে আপনার পাবলিক আইপি দেখা যায়।

নিরাপত্তা এবং সুরক্ষা

আইপি অ্যাড্রেস ওপেন থাকলে বিভিন্ন ঝুঁকি যেমন- IP Spoofing, DDoS Attacks বা Man-in-the-Middle অ্যাটাক হতে পারে।

সুরক্ষার উপায়:

  • VPN ব্যবহার করা: এটি আপনার আসল আইপি লুকিয়ে ফেলে।
  • Proxy Server: এটি আপনার ও ইন্টারনেটের মাঝে মধ্যস্থতাকারী হিসেবে কাজ করে।
  • Firewall: অননুমোদিত কানেকশন ব্লক করার জন্য ফায়ারওয়াল ব্যবহার করুন।
