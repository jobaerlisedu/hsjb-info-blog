What is OSI Model? - 7 Layers Explained

Dec 30, 2025 | Networking

Introduction

The OSI (Open Systems Interconnection) Model is a set of rules that explains how different computer systems communicate over a network. Developed by the International Organization for Standardization (ISO), it consists of 7 layers, each with specific functions and responsibilities.

The 7 Layers of the OSI Model

Layer 7: Application Layer

The topmost layer, closest to the end-user. It provides network services directly to applications.

  • Protocols: HTTP, FTP, SMTP, DNS
  • Functions: Network Virtual Terminal, File Transfer, Mail Services

Layer 6: Presentation Layer

Known as the Translation layer. Handles data formatting, encryption, and compression.

Layer 5: Session Layer

Manages sessions between applications. Establishes, maintains, and terminates connections.

Layer 4: Transport Layer

Provides end-to-end communication services for reliable data transfer (TCP/UDP).

Layer 3: Network Layer

Handles routing and forwarding of packets using IP addresses.

Layer 2: Data Link Layer

Provides node-to-node data transfer and physical addressing (MAC).

Layer 1: Physical Layer

The lowest layer, responsible for physical transmission of bits over cables/wireless.

Protocols in OSI Layers

Layer Protocol Data Unit Key Protocols
7. Application Data HTTP, FTP, DNS
6. Presentation Data SSL/TLS, JPEG
5. Session Data NetBIOS, PPTP
4. Transport Segments TCP, UDP
3. Network Packets IP, ICMP
2. Data Link Frames Ethernet, PPP
1. Physical Bits USB, Ethernet
